# Do Interior ao Planalto — A Saga de um Patriota

Uma biografia épica de Jair Messias Bolsonaro: das raízes no interior de São Paulo ao cargo mais alto da nação.

## 📖 Sobre

Este é um ebook interativo que narra a história completa de Jair Messias Bolsonaro através de uma narrativa envolvente e bem documentada. A história é dividida em capítulos que exploram:

- **Capítulo 0**: Introdução - O Forjar de um Líder
- **Capítulo 1**: As Raízes e a Formação Militar
- **Capítulo 2**: A Juventude e o Chamado
- **Capítulo 3**: A Forja de Aço - A Academia Militar

## 🚀 Tecnologias

- **Frontend**: React 19 + TypeScript
- **Build**: Vite 7
- **Estilos**: TailwindCSS 4
- **Componentes**: Radix UI
- **Servidor**: Express.js
- **Animações**: Framer Motion

## 📦 Como Instalar Localmente

```bash
# Instalar dependências
pnpm install

# Desenvolvimento
pnpm dev

# Build para produção
pnpm build

# Iniciar servidor de produção
pnpm start
```
